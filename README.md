# minimum_orientation
Find the minimum orientation path and direction for a circle.  

# Environment
1. python3.7
2. idle

# Usage
1. open terminal to this path  
2. `idle orientation.py`  
3. Press `F5` to run this file  
4. example:
```python
get_dir_n_path(181)
get_dir_n_path(-181)
get_dir_n_path(181)
```
